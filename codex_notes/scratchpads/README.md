# Scratchpads

This directory is for lower-level, mutable notes that do not belong in coordination or templates.

Good examples:

- `xsa-all.md`
- `gptq-self-calibration.md`
- `compile-safe-late-qat.md`
- `random-findings-2026-03-27.md`

Keep these concise and useful for handoff:

- branch/worktree
- commands run
- result summary
- next step
